
# Roku FlashBack APP

![RokuFlashback Logo](/images/flashback_logo.png)

## Let's take a Flashback!


RokuFlashback is a streaming app dedicated to old movies, Music and TV-Shows. 


## Built with
* **HTML5**
* **Gulp-SASS/CSS3**
* **JavaScript, Vue.js, Node.js**
* **MySQL, PHP**
